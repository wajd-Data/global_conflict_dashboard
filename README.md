

# Global Conflict Impact Dashboard (Tableau)

## Overview
This project analyzes global conflict data using Tableau, focusing on severity, geography, and the impact of external factors such as sanctions and ceasefires.

## Key Features
- Interactive map showing conflict distribution
- Time series analysis of conflict severity
- KPI panel summarizing key metrics
- Filters for year and conflict type

## Key Insights
- ~48% of conflicts involve sanctions
- Conflict severity fluctuates over time
- Certain regions experience repeated high-intensity conflicts

## Tools Used
- Tableau
- Data cleaning and calculated fields


## Note
To view the interactive dashboard, open the `.twbx` file using Tableau.
