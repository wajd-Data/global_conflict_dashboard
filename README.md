# Global Conflict Impact Dashboard (Tableau)

## Overview
This project analyzes global conflict data using Tableau, focusing on geographic distribution, time trends, and the impact of external factors such as sanctions and ceasefires.

---



---

## Key Insights
- ~48% of conflicts involve sanctions  
- Conflict severity fluctuates over time  
- Certain regions experience repeated high-intensity conflicts  

---

## Tools Used
- Tableau  
- Data Cleaning & Calculated Fields  

---

## Files
- Global_Conflict_Dashboard.twbx → Tableau file  
- Global_Conflict_Dashboard.pdf → Static dashboard  


---

## How to Use
1. Download the .twbx file  
2. Open using Tableau Desktop or Tableau Public  
3. Explore the dashboard interactively
